Evade Game

What is Evade?
-A little survival-based game built solely with Eclipse by a first time programmer.

Objective:
-Stay inside the safe-zone to accumulate as much points as possible.
 Catch the powerups.

Features:
-Time-based scoring system.
-Collision damage & Damage-Over-Time outside the safe-zone.
-The simplest (saddest) highscore-system.
-Few powerups & enemy types.
-Audiovisual masterpiece.
-Groundbreaking controls (WASD).

Instructions:
-Download the ZIP and run the Evade.jar.
